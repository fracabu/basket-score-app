<a name="english"></a>
## :gb: English

### Overview

A web-based basketball score tracking application. Features real-time scoring, player management, quarter tracking, and PDF report generation for official game records.

### Features

- **Real-Time Scoring** - Track points live
- **Player Management** - Add and manage players
- **Quarter Tracking** - Follow game progress
- **PDF Reports** - Generate official game records
- **Statistics** - Player and team stats

### Quick Start

```bash
git clone https://github.com/fracabu/basket-score-app.git
cd basket-score-app
# Open index.html in browser
```

## :it: Italiano

### Panoramica

Applicazione web per tracking punteggio basket. Real-time scoring, gestione giocatori, tracking quarti e generazione report PDF per record partite ufficiali.

### Funzionalita

- **Scoring Real-Time** - Traccia punti live
- **Gestione Giocatori** - Aggiungi e gestisci giocatori
- **Tracking Quarti** - Segui progresso partita
- **Report PDF** - Genera record partite ufficiali
- **Statistiche** - Stats giocatori e squadre

### Avvio Rapido

```bash
git clone https://github.com/fracabu/basket-score-app.git
cd basket-score-app
# Apri index.html nel browser
```
